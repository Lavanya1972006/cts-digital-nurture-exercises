## Bootstrap Folder Structure

Bootstrap is the most popular front end framework which is used for developing responsible and mobile-first website.

## Bootstrap Folder Structure

After downloading the bootstrap, the unzipped folder contains 2 folders such as css and js in the latest versions 4 and 5 but there is also another folder called fonts which was included in bootstrap version 3.In general the version 3 & 4 uses jquery but version 5 uses js.

## Css Folder

The CSS files are used to design and style the web page. They help in creating layouts, responsive design, and styling components like buttons, forms, tables, and navigation bars.The CSS folder contains many stylesheet files.

Some important files are

* bootstrap.css - Full readable CSS file
* bootstrap.min.css - Compressed CSS file (faster loading)
* bootstrap-grid.css - Grid system only
* bootstrap-reboot.css - Resets default browser styles

## Js Folder

The JS folder contains JavaScript files used for adding interactivity to web pages.These files are available in both normal and minified versions.These files are used to add interactive features like dropdown menus, popups (modals), sliders (carousel), alerts, and navigation toggle.This folder also contains many js files.In Bootstrap, plugins are included inside the JS folder itself, not as a separate folder.

Some important files are:

* bootstrap.js - Readable JS file
* bootstrap.min.js - Compressed JS file
* bootstrap.bundle.js - JS + Popper included
* bootstrap.bundle.min.js - Compressed bundle file

In Bootstrap, all plugins are included inside the JavaScript files, mainly in bootstrap.js and bootstrap.bundle.js, and there is no separate plugins folder.

## Icons

Icons are used to show small symbols like search, user, menu, and arrows to improve the user interface. Icons were included inside the package called glyphicons in older version bootstrap like Bootstrap 3 but icons are not included by default and are provided separately in bootstrap version 4 and 5.
