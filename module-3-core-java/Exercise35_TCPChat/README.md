# Exercise35 - TCP Client Server Chat (Java)

## Objective
To implement a simple TCP chat system using Java sockets.

---

## Project Structure
Exercise35_TCPChat/
- Server.java
- Client.java

---

## Compilation

javac Server.java
javac Client.java

---

## Execution

### Step 1: Run Server (Terminal 1)
java Server

### Step 2: Run Client (Terminal 2)
java Client

---

## Working
- Server waits for client connection
- Client sends messages
- Both can chat using console
- Type "exit" to stop communication

---

## Output Example
Server: Hello Client  
Client: Hi Server
